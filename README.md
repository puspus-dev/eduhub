# 🎓 EDUHub — Digitális Oktatási Portál

Egy modern, reszponzív oktatási webalkalmazás, amely az NKP inspirációjára készült — modern felhasználói élménnyel, animációkkal és egyszerű bővíthetőséggel.

## 🚀 Tech stack

- ⚛️ **React 18**
- ⚡ **Vite**
- 🎨 **TailwindCSS**
- 🌀 **Framer Motion**
- 🧩 **Lucide React Icons**
- ☁️ **Vercel** (hosting)
